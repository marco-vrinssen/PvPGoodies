# PvP Plus

WoW addon with quality-of-life features for PvP, including automatic spirit release, honor bar hiding, countdown timers on queue popups, and a copyable battleground roster.

## Features

- **Auto Release** - Automatically releases spirit when you die in battlegrounds or arenas
- **Hide Honor Bar** - Hides the honor bar in arenas and battlegrounds
- **Queue Timer** - Adds countdown timers to battleground and dungeon queue popups
- **BG Roster** - Shows a copyable list of your team's names after a battleground ends
